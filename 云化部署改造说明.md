## 本地切换到 Neon Postgres

1. 打开 `.env.local`
2. 把 Neon 的连接字符串填到：

```env
DATABASE_URL=postgresql://用户名:密码@xxx.neon.tech/neondb?sslmode=require
```

3. 执行建表：

```bash
npm run db:migrate
```

4. 启动本地服务：

```bash
npm run dev
```

如果 `DATABASE_URL` 为空，项目会继续使用本地 SQLite。

## 后续部署 Vercel 前要配置的环境变量

在 Vercel Project Settings -> Environment Variables 中设置：

```env
DATABASE_URL=Neon Postgres 连接字符串
GEMINI_API_KEY=Google Gemini API Key
GEMINI_MODEL=gemini-2.5-flash
ACCESS_PASSWORD=临时访问密码
```

后续如果做“管理员配置中心”，还会新增：

```env
SETTINGS_ENCRYPTION_KEY=用于加密配置的随机密钥
```
